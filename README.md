# Reservation Service

The Reservation Service is a project with SpringBoot API to import data from other project and do reservations.

System Requirements
- JDK 1.8
- Gradle 5.6.4
- Docker

Welcome to reservation service

### To execute

Compile the project with gradlew 

```bash
./gradlew clean build
```

After this, you can execute

```bash
docker-compose up -d
```
